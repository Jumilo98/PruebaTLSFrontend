## Pasos Iniciales para despllegar localmente el proyecto

Primero instalaremos las dependencias

```bash
npm run install
```


Luego correr el servidor local

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) desde cualquier navegador
